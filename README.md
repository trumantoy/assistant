# 微信助手

## 1.概述

### 需求

一款用于微信的自动化聊天软件，其支持的核心功能有：

- 话术
  - 一键发送话术
  - 话术编辑
- 图像
  - 一键发送图片
  - 图片编辑
- 订单
  - 一键生成订单并发送
  - 订单编辑
